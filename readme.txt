=================
OBS-SOKUHO 설정안내
=================
시작하기 앞서 OBS만 지원하는것이 아님을 알려드립니다. 
개발은 OBS를 기준으로 하였으며 설명도 OBS 기준으로 하겠습니다.
(왠만한 방송 프로그램은 될것입니다)

방송 프로그램을 킵니다.

소스목록에 브라우저를 추가합니다.

브라우저 속성에 로컬 파일 체크박스를 체크합니다.

로컬 파일 경로를 OBS-SOKUHO 폴더기준
html/index.html로 설정합니다.

아래 크기입력은
본인의 송출 해상도 크기를 권장합니다.
(매우작은 해상도가 아니라면 어느 해상도라도 자동으로 위치가 조정됩니다)

사양이 좋지 않은 컴퓨터는
사용자가 임의로 설정한 프레임 레이트 사용을 클릭하고
FPS를 낮게 설정하시는걸 추천드립니다.

OBS를 통해 오디오 조절을 체크합니다(선택사항)

CSS 입력칸을 비웁니다.

확인을 클릭합니다.

화면에 꽉차도록 조절합니다.

이제 OBS 설정이 다되었습니다. 

아래 설명에서 테스트를 진행을 안내합니다.

============
속보송출 테스트
============
OBS-SOKUHO 폴더기준 html/remote.html 을
인터넷 브라우저로 열어줍시다(브라우저는 역시 크롬보다 빠르고 가벼운 마이크로소프트 뉴엣지)

PEWS Simulation의 START 버튼을 클릭하거나
SELF Typing에서 직접 입력을하여 BROADCAST 버튼을 클릭합니다.
(HIDE 버튼은 현재 송출중인 정보를 내립니다)

화면에 잘 표시되는지 미리보기 화면에서 확인, 
경보음이 제대로 재생되고 있는지 오디오 믹서에서 확인합니다.

잘되면 끝입니다.

+ OBS-SOKUHO는 반드시 초록색 프로그램(sokuho.exe)이 켜져있는 상태에만 작동됩니다.
+ PEWS에서 신속정보 수신시 긴급지진속보를 자동송출하며 그 이외 정보는 송출하지 않습니다.
+ PEWS에서 신속정보중에 상세정보 수신시 자동으로 숨겨집니다.
